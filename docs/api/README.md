## Convenciones

- Versionado: `/api/v1/`, `/api/v2/`
- Formato: JSON, UTF-8
- Autenticación: pendiente (módulo auth)
- Paginación: `page`, `page_size`, `total`, `total_pages`
- Errores: `{ "detail": "...", "code": "..." }`
