# Proxmox Infrastructure

Configuración para despliegue en Proxmox VE.

## Planificado

- VM templates para backend/frontend
- Scripts de provisioning
- Configuración de red y firewall
- Backups y snapshots

## Requisitos

- Proxmox VE 8.x
- Docker CE en las VMs
- PostgreSQL 16 (dedicado o en container)
